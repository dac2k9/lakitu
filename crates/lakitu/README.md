# lakitu

The live TUI **cockpit** for a fleet of coordinating Claude Code agents — a
dashboard of every agent's presence, current task, inbox, personal tasks, and
the work moving through your GitHub board.

```sh
cargo install lakitu
```

This is one half of the [Lakitu](https://github.com/dac2k9/lakitu) project; it
reads the shared fleet store that the [`lakitu-mcp`](https://crates.io/crates/lakitu-mcp)
server writes. See the [project README](https://github.com/dac2k9/lakitu#readme)
for the full setup.
